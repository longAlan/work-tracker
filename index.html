SharePoint Setup

This tracker saves to a SharePoint list so your whole team sees the same data. Complete these steps once:

  1. Go to your SharePoint site → Site contents → New → List
  2. Name it exactly: WorkTracker
  3. Inside the list, click Add column → Multiple lines of text
  4. Name the column exactly: TrackerData → Save
  5. Upload this HTML file to Site Assets and open it from there
  6. Paste your SharePoint site URL below and click Save
Site URL format: https://yourcompany.sharepoint.com/sites/yoursite
No trailing slash. This is saved in your browser for convenience.
