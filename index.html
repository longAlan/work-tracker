JSONBin Setup

This tracker saves to JSONBin so your whole team shares the same data. Complete once:

  1. Go to jsonbin.io and create a free account
  2. Click API Keys in the left sidebar
  3. Copy your Master Key and paste it below
  4. Click Save & Connect — a shared bin is created automatically on first save
  5. Share the Bin ID (shown after first save) with teammates so they can connect to the same data
Teammates paste the same Master Key + Bin ID to connect to your shared tracker. The Bin ID appears here after the first save.
